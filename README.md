# Kratix (kratix)
Kratix is an open-source framework for building internal developer platforms using Kubernetes, enabling platform teams to deliver customizable, composable Promises as a Service. Kratix exposes its capabilities through Kubernetes Custom Resource Definitions (Promise, Resource, WorkloadGroup, Destination, StateStore) rather than a traditional HTTP API.
